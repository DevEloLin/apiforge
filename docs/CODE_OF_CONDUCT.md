# Code of Conduct / 行为准则

## English

This project adopts the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/)
v2.1. Be respectful, welcoming, and constructive. Harassment, discrimination, or abusive behavior
of any kind is not tolerated.

Report unacceptable behavior privately to **dev.elolin@gmail.com**. Maintainers may remove,
edit, or reject contributions and comments that violate this code, and may ban offenders.

## 中文

本项目遵循 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 的精神：
互相尊重、友善、建设性。不容忍任何形式的骚扰、歧视或辱骂行为。

如遇不当行为，请私下发邮件到 **dev.elolin@gmail.com** 举报。维护者有权移除、修改或拒绝违反
本准则的贡献与评论，并可封禁违规者。
